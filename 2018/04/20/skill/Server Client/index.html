COMSOL Server及Client使用

By aihkoo

发表于 Apr 20 2018

Server初始化

本周终于靠着三千预算解决了只差一个服务器的问题，然后开始了漫长的COMSOL Server配置过程。

首先参照COMSOL 系统要求安装两个基础包libwebkitgtk-1.0-0和lsb-core

sudo apt-get install libwebkitgtk-1.0-0 lsb-core

然后就可以按照之前的COMSOL Server配置安装COMSOL Server和初始化配置，就在我以为大功告成之际，残酷的现实告诉我，不肯花钱是要付出时间的。

以下省略一个星期的Baidu和Google时间，因为真是Baidu无门，Google无路。最后发现的问题和解决方案都很简单，5.3a SSQ有bug，滚回5.3。

Client登陆和使用

COMSOL支持两种登陆方式，网页登陆和Client登陆。

网页登陆

直接输入Server地址加端口号，中间用冒号隔开，即可登陆。网页支持管理员账户登陆，管理员可查看Server运行状态，管理其他用户等。

Client登陆

Client需要在官网单独下载COMSOL Client，打开后在Server处输入地址，右侧输入用户名密码即可登陆。

Client和网页端显示基本一样。需要运行仿真时选择Upload，点击Choose applications。选择要仿真的程序并点击Upload，等待上传完成，然后找到上传的仿真程序，Run！
