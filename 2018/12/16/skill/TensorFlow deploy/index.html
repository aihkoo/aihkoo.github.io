我要炼丹之 Win10 TensorFlow(GPU) 安装

By aihkoo

发表于 Dec 16 2018

环境准备

硬件环境 E5 2660v2*2 + GTX 1080Ti
操作系统 Windows 10 x64
软件列表 Anaconda3 + CUDA 10.0 + cuDNN v7.4 + Tensorflow-GPU 1.12
开发环境 pycharm

安装顺序

安装 NVIDIA CUDA驱动
下载CUDA和cuDNN
安装CUDA
cuda安装完成后默认的环境变量配置CUDA_PATH是C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.0

但是这样不能直接访问到bin和lib\x64下的程序包

在path中加上这两个路径：
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.0\bin
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.0\lib\x64

安装cuDNN
拷文件

安装Anaconda3

tensorflow环境
conda create -n tensorflow python=3.6
activate tensorflow
pip install tensorflow-gpu

启动TensoFlow环境
activate tensorflow
关闭
deactivate

安装pycharm
在pycharm解析器中选择tensorflow环境python3.6

基本程序

import tensorflow as tf
hello = tf.container(‘Hello, TensorFlow!’)
print(tf.version)
